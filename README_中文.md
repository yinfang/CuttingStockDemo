# 一维下料问题求解器

本应用程序实现了一个改进的混合遗传算法，用于解决一维下料问题。

## 功能特性

- 为下料问题实现混合遗传算法
- WPF用户界面用于数据输入和可视化
- Excel导出功能用于解决方案结果
- 支持多种原材料长度和切割项目规格

## 项目结构

```
CuttingStockProblem/
├── src/                 # 主源代码

```

## 安装说明

1. 安装 .NET 8.0 SDK
2. 安装 Visual Studio 或 Visual Studio Code
3. 在您首选的IDE中打开解决方案
4. 构建并运行应用程序

## 使用方法

1. 在数据输入选项卡中指定原材料和切割项目
2. 在算法设置选项卡中配置算法参数
3. 运行算法生成切割模式
4. 将结果导出到Excel进行进一步分析

## 依赖项

- .NET 6.0
- ClosedXML (用于Excel导出功能)
- WPF (用于用户界面)

## 算法概述

遗传算法实现包括：
- 随机解决方案的种群初始化
- 适应度计算以评估解决方案质量
- 父代选择机制
- 交叉和变异操作
- 精英主义保留最佳解决方案
- 多目标优化以最小化浪费和使用的原材料
