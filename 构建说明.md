# 构建说明

## 先决条件

要构建此项目，您需要安装.NET 6.0 SDK。

## 安装.NET 6.0 SDK

1. 访问 https://dotnet.microsoft.com/download/dotnet/6.0
2. 下载并安装.NET 6.0 SDK（不是仅运行时）

## 或者使用全局安装脚本

您可以使用以下脚本安装.NET SDK：

### Windows (使用Chocolatey)
```powershell
choco install dotnet-6.0-sdk
```

### Windows (使用Webi)
```powershell
curl -s https://webinstall.dev/dotnet | bash
```

## 构建项目

安装.NET 6.0 SDK后，您可以使用以下命令构建项目：

```cmd
dotnet build
```

或者运行现有的构建脚本：
```cmd
build.bat
```

## 运行项目

构建成功后，您可以使用以下命令运行应用程序：
```cmd
dotnet run --project src/CuttingStockProblem.csproj
```
