# 一维下料问题求解器 - 项目总结报告

## 项目概述

一维下料问题求解器是一个使用遗传算法解决一维材料切割优化问题的WPF应用程序。该应用程序旨在帮助制造业、建筑业等需要精确切割材料的行业优化材料使用，减少浪费，提高生产效率。

### 核心功能
1. **遗传算法优化**：使用遗传算法解决一维下料问题
2. **数据输入**：支持输入原材料和切割需求
3. **参数配置**：可配置遗传算法参数
4. **结果展示**：清晰展示优化结果
5. **Excel导出**：支持将结果导出到Excel

## 使用说明

### 系统要求
- .NET 6.0 SDK 或更高版本
- Windows操作系统

### 安装和运行
1. 确保已安装.NET 6.0 SDK
2. 克隆或下载项目源代码
3. 在项目根目录下运行：
   ```
   dotnet build
   ```
4. 运行测试：
   ```
   dotnet test
   ```

### 界面功能
应用程序主界面分为三个主要标签页：

1. **数据输入标签页**
   - 原材料管理（长度、数量、成本）
   - 切割项目管理（长度、需求数量）

2. **算法设置标签页**
   - 种群大小设置
   - 最大迭代次数设置
   - 交叉率和变异率设置
   - 损耗值设置

3. **结果展示标签页**
   - 解决方案摘要（总模式数、总浪费、效率、总成本）
   - 详细结果展示
   - 导出功能
