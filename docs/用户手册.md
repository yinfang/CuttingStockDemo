# 一维下料问题求解器用户手册

## 简介

一维下料问题求解器是一个专门用于解决一维材料切割优化问题的软件工具。它可以帮助制造业、建筑业等需要精确切割材料的行业优化材料使用，减少浪费，降低成本。

## 系统要求

- 操作系统：Windows 7 或更高版本
- .NET 6.0 运行时环境
- 至少 2GB 可用内存
- 至少 100MB 可用磁盘空间

## 安装说明

1. 下载安装包
2. 双击安装文件开始安装
3. 按照安装向导完成安装过程
4. 启动应用程序

## 界面介绍

应用程序主界面分为三个主要标签页：

### 1. 数据输入标签页

此标签页用于输入原材料信息和切割需求。

#### 原材料管理
- **长度**：原材料的标准长度
- **数量**：可用的原材料数量
- **成本**：每单位原材料的成本

#### 切割项目管理
- **长度**：需要切割的长度
- **数量**：需要切割的数量
- **需求**：项目的需求次数

#### 操作按钮
- **添加原材料**：添加新的原材料记录
- **删除原材料**：删除选中的原材料记录
- **添加项目**：添加新的切割项目
- **删除项目**：删除选中的切割项目

### 2. 算法设置标签页

此标签页用于配置遗传算法的参数。

#### 算法参数
- **种群大小**：遗传算法种群中个体的数量（默认：100）
- **最大迭代次数**：算法运行的最大代数（默认：500）
- **交叉率**：两个个体交叉的概率（默认：0.8）
- **变异率**：个体发生变异的概率（默认：0.1）
- **损耗值**：切割时的最小损耗（默认：0.1）

#### 运行算法
点击"运行算法"按钮开始优化计算。

### 3. 结果展示标签页

此标签页显示优化计算的结果。

#### 解决方案摘要
- **总模式数**：生成的切割模式总数
- **总浪费**：所有切割模式的总浪费量
- **效率**：材料使用效率百分比
- **总成本**：总材料成本

#### 详细结果
显示每个切割模式的详细信息：
- 模式编号
- 原材料长度
- 项目数量
- 浪费量

#### 操作按钮
- **导出到Excel**：将结果导出到Excel文件
- **打印解决方案**：打印当前解决方案

## 使用流程

### 1. 输入数据
1. 在"数据输入"标签页中输入原材料信息
2. 输入切割项目需求
3. 确保所有数据输入正确

### 2. 配置算法
1. 切换到"算法设置"标签页
2. 根据需要调整算法参数
3. 点击"运行算法"按钮开始计算

### 3. 查看结果
1. 切换到"结果展示"标签页
2. 查看解决方案摘要
3. 查看详细结果信息

### 4. 导出结果
1. 点击"导出到Excel"按钮
2. 选择保存位置
3. 保存Excel文件

## 常见问题解答

### 1. 算法运行时间过长怎么办？
可以尝试以下方法：
- 减少种群大小
- 减少最大迭代次数
- 检查输入数据是否合理

### 2. 结果不理想怎么办？
可以尝试以下方法：
- 调整算法参数
- 检查输入数据的准确性
- 增加迭代次数

### 3. 无法导出到Excel怎么办？
请检查：
- 是否已安装Excel
- 是否有文件写入权限
- 磁盘空间是否充足
