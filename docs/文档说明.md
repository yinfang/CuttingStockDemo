# Cutting Stock Problem Solver Documentation

## Overview

This application implements an improved hybrid genetic algorithm for solving the one-dimensional cutting stock problem. The algorithm is designed to minimize waste and optimize the use of stock materials when cutting items of various lengths.

## Features

1. **Genetic Algorithm Implementation**: Uses an improved hybrid genetic algorithm based on the research paper
2. **WPF User Interface**: Modern graphical interface for data input and visualization
3. **Excel Export**: Export solutions to Excel format for further analysis
4. **Multi-stock Support**: Handle multiple stock lengths and types

## Algorithm Details

The genetic algorithm implementation includes:

- Population initialization with random solutions
- Fitness calculation to evaluate solution quality
- Selection mechanisms for parent selection
- Crossover and mutation operations
- Elitism to preserve best solutions
- Multi-objective optimization to minimize waste and stocks used

## Installation

1. Install .NET 6.0 SDK
2. Install Visual Studio 2022 or later
3. Open the solution file in Visual Studio
4. Build and run the application

## Usage

1. **Input Data Tab**: 
   - Add stock materials with length, quantity, and cost
   - Add cutting items with length and quantity requirements
   - Use the "Add Stock" and "Add Item" buttons to add new entries
   - Use the "Remove Stock" and "Remove Item" buttons to remove entries

2. **Algorithm Settings Tab**:
   - Adjust algorithm parameters:
     - Population Size: Number of individuals in the population
     - Max Generations: Maximum number of generations for evolution
     - Crossover Rate: Probability of crossover operation
     - Mutation Rate: Probability of mutation operation
     - Trim Loss: Minimum material that must be left when cutting

3. **Results Tab**:
   - View solution summary with total patterns, waste, efficiency, and cost
   - See detailed cutting patterns
   - Export results to Excel

## Model Classes

### CuttingItem
Represents an item that needs to be cut with properties for length, quantity, and demand.

### StockMaterial
Represents stock material with properties for length, quantity, and cost.

### CuttingPattern
Represents a cutting pattern solution containing items cut from a stock material.

### SolutionResult
Represents the final cutting solution with patterns, waste, and efficiency metrics.

## Algorithm Implementation

The genetic algorithm follows these steps:

1. **Initialization**: Create initial population of random solutions
2. **Evaluation**: Calculate fitness for each individual solution
3. **Selection**: Select parents for reproduction based on fitness
4. **Crossover**: Combine parent solutions to create offspring
5. **Mutation**: Introduce random changes to maintain diversity
6. **Replacement**: Create new population for next generation

## Excel Export

The application can export solutions to Excel format including:
- Solution summary with waste, efficiency, and cost metrics
- Detailed cutting patterns
- Item requirements and stock usage
